# Veya Analytics SDK Reference

A searchable reference for all trackable metrics and events in the Veya Analytics SDK.

## Features

- **Search** - Filter metrics by name, method, or description
- **Category filters** - Filter by event category (Session, E-commerce, Menu, Search, etc.)
- **Dark/Light mode** - Toggle between themes
- **Responsive** - Works on desktop and mobile

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Deployment

Optimized for Vercel. Connect your GitHub repo and deploy automatically.

## Event Categories

| Category | Description |
|----------|-------------|
| Session | Page visibility, session start/end |
| Navigation | Page views, scroll depth, clicks |
| E-commerce | Cart, checkout, purchase funnel |
| Authentication | Login, logout, identify |
| Store | Store selection, order type |
| Menu | Menu browsing, category navigation |
| Search | Search queries, results, clicks |
| Loyalty | Promos and rewards |
| Error | Application errors |
| Experiments | A/B test assignments |
